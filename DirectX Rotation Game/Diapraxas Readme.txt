OVERVIEW
---------------------------------
Rotation is a puzzle game where the goal
is to arrange bricks in certain patterns.
5 levels are available until the game is
registered.
See web site for more information about
registration and game updates.


HOW TO PLAY
---------------------------------
You move the balls clock-wise or
counter-clock-wise by pressing 
left or right mouse button.
Each level is completed by arranging
the bricks in the middle like the pattern
in the upper right corner.
In the registered version there is a Custom
game mode available which can create random
games.


We hope you will enjoy the game!
